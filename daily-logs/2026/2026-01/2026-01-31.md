## ðŸ’¡ What I Learned

### Technical
- ** Mods:** A way for the prompt to be enriched before the editor sees it, and the response to be improved before it appears to the editor.
- Mods improves by:
    - **Injecting context:** mod adding more information to the prompt before the agent sees it.
    - **Providing tools:** giving the agent super powers, such as fetching rust crates & sources, and cargo tests & build.
    - **Transforming output:** modifying the output before it appears in the editor to improve the quality of the code generated by the agent.
